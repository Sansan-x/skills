## 来源详情

### SRC-001: 华为Go白盒安全测试指导

**类型：** security-guide  
**文档标题：** GO白盒安全测试指导 V1.0  
**发布单位：** 华为技术有限公司  

**主要章节覆盖：**
- 4.1 危险函数 (unsafe库、命令注入)
- 4.2 panic (数组越界、空指针、除零)
- 4.3 并发安全 (条件竞争)
- 4.4 throw (内存耗尽)
- 4.5 敏感信息 (硬编码、安全随机数)
- 4.6 整数安全
- 4.7 Web (SQL注入、XSS、命令注入、SSRF、模板注入)
- 4.8 其他 (ReDoS)
- 4.9 CGO

**提取的模式类别：**
- SQL注入 (SQLI)
- 命令注入 (CMDI)
- XSS
- SSRF
- 敏感信息 (INFO)
- 并发安全 (RACE, GOLNG)
- 密码学失败 (CRYPTO)

**测试方法论贡献：**
- 自外向内数据流分析
- 自内向外关键字搜索
- 工具使用指导 (GoLand, PowerGrep, Seninfo)
